void CAN_CheckInvalidInit ( void  )

The test case CAN_CheckInvalidInit verifies the driver behaviour when receiving an invalid initialization sequence:

  • Uninitialize
  • PowerControl with Power off
  • PowerControl with Power on
  • Set Mode
  • PowerControl with Power off
  • Uninitialize
