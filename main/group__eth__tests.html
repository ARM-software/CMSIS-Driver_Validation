void ETH_PHY_PowerControl ( void )

Function: ETH_PHY_PowerControl.

The function ETH_PHY_PowerControl verifies the Ethernet PHY PowerControl function with the sequence:

  • Initialize
  • Power on
  • Power low
  • Power off
  • Check RMII lockup
  • Uninitialize
Note
When the Ethernet PHY provides the 50 MHz reference clock for the MAC in RMII mode, the MAC may lock up if the PHY is powered down and no clock is generated.
