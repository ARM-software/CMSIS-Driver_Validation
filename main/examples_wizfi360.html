WIZnet WizFi360-EVB WiFi Shield with NXP MIMXRT1064-EVK

Software Setup

Using the Pack Installer install the latest NXP::MIMXRT1064_DFP and NXP::EVK-MIMXRT1064_BSP packs and copy the example project CMSIS-Driver WiFi WIZnet WizFi360 Validation (EVK-MIMXRT1064) to your machine.

  1. Choose one of the available Project Targets and build the project.
  2. Run the validation on the target hardware using the on-board CMSIS-DAP Debugger.

This example is prepared for verification of the WiFi driver and it requires WiFi requirements, as well as, proper configuration described in WiFi Configuration.

For details on WiFi driver tests please refer to WiFi Validation.
