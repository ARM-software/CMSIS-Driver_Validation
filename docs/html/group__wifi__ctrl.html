void WIFI_Initialize_Uninitialize ( void  )

The test case WIFI_Initialize_Uninitialize verifies the WiFi Driver Initialize and Uninitialize functions.

int32_t (*Initialize) (ARM_WIFI_SignalEvent_t cb_event);

and

int32_t (*Uninitialize) (void);

Testing sequence:

  • Initialize without callback
  • Uninitialize
  • Initialize with callback (if driver supports it)
  • Power on
  • Uninitialize
  • Initialize without callback
  • Power on
  • Power off
  • Uninitialize
